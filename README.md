
Projetinho de iniciativa propria.

Desenvolvido com React utilizando hooks (useState, useEffect, useParams, useContext) além disso também utizo o router-dom para navegação entre páginas, axios para conexão api e também Fetch, a ideia era ir testando mesmo, por isso utilizei dois metodos de conexão diferente.

Para trazer o catálogo de series e filmes é feito uma integração com a API do ThemovieDb.

Inicialmente a ideia era construir apenas a home, mas me animei e fui fazendo algumas implementações, ainda tem coisas legais para ser implementado, sempre que sobrar um tempinho vou voltando e melhorando o projeto, por isso se chegar em algum ponto onde ainda esteja "muito bruto" tenha em mente que é projeto em andamento e não finalizado ;)
